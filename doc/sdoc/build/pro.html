Проект язык программирования Ukuvchi(узб.-ученик)¶

  • Компилятор написан на Python3(тестировался на Python3.6.3)
  • исходный текст идет в lisp подобном синтаксисе,идет интерпритация так называемых s(sybolic) выражений.
  • Нет массивов,строк,структур и т.д.
  • Язык служит для демонстрационных целей,того как может работать компилятор и интерпритатор байт-кода(виртуальная машина),котарая написана на С, как расширение для Python3,библиотека с расширением pyd.
  • Для перемнных используются толко буквы a-y,буква z чтобы снимать с регистра,то что вернула прошлая функция

Документация для байт-интерпритатора https://our-ruzaevka.herokuapp.com/rest/ddoc/index.html
